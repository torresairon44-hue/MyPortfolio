This is a recruiter-focused, single-page portfolio built with HTML, CSS, and JavaScript.

## How to Use

- Open index.html in a browser.
- Edit index.html to update content.
- Replace placeholder contact details and links.
- Add your resume file as resume.pdf in the project root to enable download and preview.
- Add photos and certificates into the assets folder (see below).

## Photos & Certificates

Place your images in the assets folder and update filenames if needed:

- profile photo: assets/profile.jpg
- gallery photos: assets/gallery-1.jpg through assets/gallery-4.jpg
- certificates: assets/cert-1.jpg through assets/cert-4.jpg

## Customize

- Colors and layout: styles.css
- Dark/light toggle and small interactions: script.js
